# 계정 및 데이터 삭제 요청 안내 (Account & Data Deletion)

**"왕의 길(The King's Way)"**는 사용자의 개인정보 보호와 데이터 주권을 존중합니다. 구글 플레이 스토어의 데이터 안전 정책에 따라, 계정 및 관련 데이터의 삭제를 요청하는 방법을 아래와 같이 안내해 드립니다.

## 1. 삭제되는 데이터의 범위
계정 삭제 요청 시, 다음의 데이터가 즉시 또는 지정된 보유 기간 후 영구적으로 삭제됩니다.
- **계정 정보**: 이메일 주소, 사용자 ID, 소셜 로그인 연동 정보.
- **게임 진척도**: 현재 날짜(Day), 읽은 책 목록, 달성한 업적(품계) 정보.
- **활동 기록**: 조강(朝講) 완료 내역, 어전회의 결정 사항, 국정 운영 로그.
- **메타데이터**: 사용자 커스텀 설정(글자 크기, 음량 등).

## 2. 계정 삭제 요청 방법

### 방법 A: 앱 내에서 즉시 삭제 (추천)
앱이 설치되어 있는 경우 가장 빠르고 확실한 방법입니다.
1. 앱을 실행하고 메인 화면(서고) 우측 상단의 **설정(톱니바퀴 아이콘)**을 클릭합니다.
2. 설정 하단의 **'계정 탈퇴'** 메뉴를 클릭합니다.
3. 데이터 삭제에 대한 안내를 확인한 후 **'탈퇴 확인'**을 선택하면 즉시 모든 데이터가 삭제되고 로그아웃됩니다.

### 방법 B: 웹/이메일을 통한 삭제 요청
앱을 이미 삭제했거나 접근이 어려운 경우 아래 방법을 이용하실 수 있습니다.
- **GitHub Issue**: 본 저장소의 [Issues](https://github.com/ChaPDCha/the-kingsway-privacy-policy/issues) 탭에 '계정 삭제 요청' 제목으로 요청을 남겨주세요. (개인정보 보호를 위해 상세 메일 주소는 적지 마세요.)
- **개발자 메일**: 앱 스토어에 등록된 개발자 연락처로 계정(이메일)과 함께 삭제 요청을 보내주시면 본인 확인 후 7일 이내에 조치해 드립니다.

## 3. 데이터 보관 정책
- 사용자가 삭제를 요청하면 시스템은 해당 데이터를 즉시 파기하는 것을 원칙으로 합니다.
- 단, 관련 법령에 의해 보존할 의무가 있는 데이터(결제 내역 등)는 법령이 정한 기간 동안 별도 보관 후 파기됩니다.
- 삭제된 데이터는 복구가 불가능하오니 신중하게 결정해 주시기 바랍니다.
