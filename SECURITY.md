# Security Policy

## Supported Versions

The following versions of **The King's Way** are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We prioritize the security of our users and their data. If you discover a security vulnerability within this project or the associated application, please let us know immediately.

### How to report
- **Email**: Please report vulnerabilities by emailing [chapdcha@gmail.com](mailto:chapdcha@gmail.com).
- **Do not use GitHub Issues**: To prevent the vulnerability from being publicly exploited before a fix is available, please *do not* create a public Issue for security-related bugs.

### Our Commitment
- We will acknowledge receipt of your report within 48 hours.
- We will provide an estimated timeline for a fix and keep you updated on our progress.
- Once the vulnerability is resolved, we will provide credit to the reporter (if desired) in our release notes.

Thank you for helping keep **The King's Way** secure for everyone.
